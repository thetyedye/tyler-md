## Patterns I keep coming back to

- **Files over databases** when files work. Markdown + `grep` beats embeddings for most personal retrieval.
- **Thin harness, fat skills.** Decision logic lives in readable markdown skills, not buried in code.
- **Brain-first.** Check the local brain before reaching for an external API.
- **Source-backed output.** Cite the file or the dataset, or say "I don't know."
